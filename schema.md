# Описание базы данных

## Таблица: `[post_data]`
**Бизнес-смысл:** [полученные данные с формы для учета выполненных задач].

```sql
CREATE TABLE [впиши_имя_таблицы] (
    number INT PRIMARY KEY,      -- Уникальный системный ID записи (Auto Increment)
    id INT,                      -- ВНИМАНИЕ: это ID [пользователя / клиента / заказа — НАПИШИ НУЖНОЕ], а не Primary Key!
    date VARCHAR(255),           -- КРИТИЧЕСКИ ВАЖНО: дата хранится как текст! Пример формата: "2026-05-24 14:20". При фильтрации всегда делай явный STR_TO_DATE / CAST.
    city TEXT,                   -- Город (например: "Москва")
    url_photo TEXT,              -- URL-ссылка на прикрепленное фото
    action TEXT,                 -- Тип события/действия
    answer TEXT,                 -- Текст ответа
    comment TEXT                 -- Комментарий оператора/пользователя
);
